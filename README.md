# Implementing_Graph_Algorithms
Exercise:
Using the directed, weighted graph shown in the figure (nodes S, B(top), B(bottom), C, G, E, F), write a single Python program that 
**(1)**	represents this graph in code (adjacency list)
**(2)**	implements DFS and BFS to find any valid path from S to G by treating every edge as unweighted (ignore weights for DFS/BFS)
**(3)**	implements A* to find the minimum-cost path from S to G using the given edge weights and a heuristic function you define and justify (e.g., h(n)=0 to reduce to Dijkstra, or a simple admissible estimate based on node layout)
**(4)**	prints for each algorithm the final path and the number of expanded nodes (and for A* also print the total path cost)
**(5)**	writes a short comparison that clearly differentiates DFS vs BFS vs A* in terms of data structure used (stack/queue/priority queue), completeness, optimality, time complexity, space complexity, and when each should be used in real applications.
